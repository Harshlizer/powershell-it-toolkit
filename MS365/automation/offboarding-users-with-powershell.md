# Offboarding Users with PowerShell

## Overview

A single PowerShell workflow can reduce manual work and keep Microsoft 365 offboarding consistent.

## Typical Actions

- Disable the account
- Reset the password
- Remove licenses
- Remove group membership and admin roles
- Hide the mailbox from address lists
- Remove aliases and inbox rules
- Wipe mobile devices
- Revoke sessions
- Convert the mailbox to a shared mailbox
