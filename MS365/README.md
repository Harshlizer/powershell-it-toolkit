# MS365

English Microsoft 365 knowledge base exported from Notion.

This section contains translated notes, operational guidance, and sanitized scripts related to Microsoft 365, Entra ID, Defender, Teams, and automation scenarios.

Sensitive values such as tenant IDs, client IDs, client secrets, SMTP credentials, webhook tokens, and similar identifiers were replaced with placeholders.

## Structure

- `automation/` - workflow and process automation notes
- `identity/` - Entra ID, Conditional Access, passkeys, and authentication topics
- `integration/` - service integration notes
- `scripts/` - sanitized PowerShell examples
- `security/` - Microsoft Defender and phishing-related topics
- `teams/` - Microsoft Teams governance and collaboration topics
