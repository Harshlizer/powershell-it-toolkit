# MS Teams Bot Integration

## Purpose

This integration pattern uses Microsoft Teams for outbound notifications and inbound AI-assisted conversations.

## Required Configuration

```json
{
  "tenant_id": "<TENANT_ID>",
  "client_id": "<CLIENT_ID>",
  "client_secret": "<CLIENT_SECRET>",
  "teams_app_id": "<TEAMS_APP_ID>",
  "service_url": "<BOT_SERVICE_URL>"
}
```
