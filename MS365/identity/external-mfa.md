# Microsoft Entra External MFA

## Overview

Microsoft Entra ID now supports external MFA providers in a way that preserves Conditional Access and risk-based controls.

## Best Fit Scenarios

- Mergers and acquisitions
- Legacy identity environments
- Compliance-driven exceptions
- Enterprises with phased modernization plans

## Important Planning Note

Microsoft Custom Controls are scheduled for retirement on September 30, 2026. Environments that still depend on them should start transition planning early.
