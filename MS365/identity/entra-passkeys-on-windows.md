# Microsoft Entra Passkeys on Windows

## Overview

Microsoft Entra passkeys on Windows provide a phishing-resistant authentication option backed by Windows Hello.

## Key Points

- Supports face, fingerprint, and PIN through Windows Hello
- Useful for devices that are not Entra joined or Entra registered
- Helpful for BYOD, shared devices, and unmanaged endpoints
- Passkeys are device-bound and do not sync across devices
- This is not a replacement for Windows Hello for Business sign-in to Windows itself
