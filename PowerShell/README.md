# PowerShell Knowledge Base

This folder contains the English version of the PowerShell materials migrated from the internal Notion knowledge base.

## Security and Sanitization

- All Russian text was translated to English.
- All passwords, login names, usernames, tenant IDs, app IDs, client secrets, SMTP credentials, and similar sensitive values were replaced with placeholders.
- Replace values such as `<TENANT_ID>`, `<CLIENT_ID>`, `<CLIENT_SECRET>`, `<USERNAME>`, `<PASSWORD>`, `<DOMAIN>`, and `<EMAIL_ADDRESS>` before use.

## Structure

- `catalog/` - scripts imported from the PowerShell scripts catalog
- `dns/` - DNS and connectivity checks
- `exchange/` - Exchange Online and mail forwarding utilities
- `monitoring/` - reporting, auditing, and alerting scripts
- `ms365/` - Microsoft 365 and Graph-based reporting scripts
- `system/` - local system administration and cleanup scripts
- `utilities/` - general-purpose command collections and quick helpers
